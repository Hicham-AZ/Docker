# Petite Application Node.js

Une petite application web simple utilisant Express.js.

## Installation

1. Cloner ou télécharger le projet.
2. Exécuter `npm install` pour installer les dépendances.

## Exécution

- Localement : `npm start`
- Avec Docker : `docker build -t petite-app .` puis `docker run -p 3000:3000 petite-app`

L'application sera accessible sur http://localhost:3000
